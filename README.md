# Ndifreke-Abasi Bassey Portfolio

A futuristic, animated personal portfolio built with React, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Personalization checklist

- Replace placeholder social links in `src/data/portfolioData.js`
- Add a profile photo or logo in `public/`
- Drop your CV into `public/Ndifreke-Abasi-Bassey-CV.pdf`
- Update demo and GitHub links for projects
- Replace placeholder testimonials with real client or peer feedback
