# Ndifreke-Abasi Bassey Portfolio

[![Live Demo](https://img.shields.io/badge/Live%20Demo-ndifreke--dev.netlify.app-blue)](https://ndifreke-dev.netlify.app/)

A modern, futuristic personal portfolio website showcasing my skills, projects, and services as a full-stack developer. Built with React, Tailwind CSS, and Framer Motion for smooth animations and responsive design.

## 🚀 Features

- **Responsive Design**: Optimized for all devices and screen sizes
- **Smooth Animations**: Powered by Framer Motion for engaging user experience
- **Dark Theme**: Sleek dark aesthetic with glassmorphism effects
- **Interactive Components**: Hover effects, scroll animations, and dynamic content
- **Contact Form**: Integrated with Netlify Forms for easy communication
- **SEO Friendly**: Proper meta tags and semantic HTML

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Netlify
- **Forms**: Netlify Forms

## 🏃‍♂️ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ndifreke-bassey/my-portfolio.git
   cd my-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 📦 Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.

## 🚀 Deployment

This project is configured for easy deployment on Netlify:

1. Connect your GitHub repository to Netlify
2. Set build command: `npm run build`
3. Set publish directory: `dist`
4. Enable Netlify Forms for the contact form

## 📝 Customization

To personalize this portfolio:

1. Update personal information in `src/data/portfolioData.js`
2. Replace placeholder images in `public/`
3. Add your CV as `public/Ndifreke-Abasi-Bassey-CV.pdf`
4. Update social links and project URLs
5. Replace testimonials with real ones
